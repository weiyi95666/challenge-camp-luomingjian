# MCP Agent 快速使用指南

## 🚀 最简单的使用方式

### 方式一：双击运行（推荐）
直接在文件夹中双击以下文件之一：
- `start_all.bat` - 一键启动所有服务
- `run_desktop.bat` - 只启动桌面聊天应用
- `run_client.bat` - 只启动命令行客户端

### 方式二：使用 PowerShell 启动器
右键点击 `run.ps1`，选择"使用 PowerShell 运行"，然后选择您想要的模式。

---

## 💻 命令行使用（需要在正确的目录下）

### 重要：首先进入项目目录
```powershell
cd "c:\Users\weiyi\Desktop\challenge-camp-luomingjian-main\MCP Agent"
```

### 然后运行您需要的功能：

#### 1. 运行 CLI 客户端
```powershell
.\.venv\Scripts\python.exe client.py "天气 上海"
```

#### 2. 运行桌面聊天
```powershell
.\.venv\Scripts\python.exe desktop_chat.py
```

#### 3. 运行 API 服务器
```powershell
.\.venv\Scripts\python.exe api_server.py
```
