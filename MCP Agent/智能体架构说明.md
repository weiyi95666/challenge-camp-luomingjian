## 📦 新增的核心模块

### 1. **agent/memory.py** - 对话记忆
- 管理对话历史
- 支持上下文记忆
- 可限制历史长度

### 2. **agent/tool_selector.py** - 智能工具选择
- 规则基础的工具选择
- 为 LLM 驱动的选择预留接口
- 工具参数自动解析

### 3. **agent/llm_client.py** - LLM 客户端
- **为您的 API 预留的集成位置**
- 支持真实 LLM 调用
- 包含模拟响应用于开发测试

### 4. **agent/core.py** - 智能体核心
- 整合所有组件
- 统一的处理流程
- 单例模式管理

### 5. **config.py** - 配置管理
- 环境变量加载
- API 配置管理
- 灵活的参数设置

## 🎯 如何集成您的 API

### 步骤 1: 提供 API 信息

请告诉我您的 API 的以下信息：

```
1. API URL (例如: https://api.openai.com/v1/chat/completions)
2. API Key (或认证方式)
3. 请求格式 (JSON body 的结构)
4. 响应格式 (如何提取回复内容)
5. 是否支持工具调用 (Function calling)
```

### 步骤 2: 我会帮您集成

一旦您提供了 API 信息，我会：

1. 更新 `agent/llm_client.py` 中的 `chat()` 方法
2. 更新 `agent/core.py` 中的 `_generate_with_llm()` 方法
3. 测试整个流程确保正常工作

### 步骤 3: 配置 `.env` 文件

我会帮您更新 `.env` 文件，填入您的 API 配置：

```env
LLM_API_URL=您的API地址
LLM_API_KEY=您的API密钥
MODEL_NAME=您的模型名称
```

## 💡 智能体的工作流程

```
1. 用户输入
   ↓
2. 存入记忆
   ↓
3. 选择工具 (LLM 或规则)
   ↓
4. 调用工具 (如需要)
   ↓
5. 生成回复 (使用 LLM)
   ↓
6. 存入记忆并返回
```
