## 📡 API 接口文档

### 聊天接口
- **URL**：`POST http://127.0.0.1:5001/chat`
- **请求体**：
  ```json
  {
    "text": "天气 上海"
  }
  ```
- **响应**：
  ```json
  {
    "response": "智能体的回复",
    "tool_used": "使用的工具名称",
    "tool_result": "工具执行结果"
  }
  ```

### 历史记录接口
- **URL**：`GET http://127.0.0.1:5001/history`

### 清空记忆接口
- **URL**：`POST http://127.0.0.1:5001/clear`
